# Documentation

This directory contains technical documentation and development notes for the Acordes MIDI Piano TUI Application.

## Files

### Setup Guides
- **`MACOS_SETUP.md`**: macOS-specific installation and setup instructions

### Synth Mode Development Notes
- **`PERFORMANCE_OPTIMIZATIONS.md`**: Details of audio performance optimizations in the synthesizer engine
- **`MIDI_SYNCHRONIZATION.md`**: Technical explanation of MIDI event queue synchronization system
- **`TESTING_MIDI_SYNC.md`**: Testing procedures for MIDI synchronization features

### UI Design Documentation
- **`UI_BOX_LAYOUT.md`**: ASCII box-drawing character specifications and layout design
- **`UI_REDESIGN_SUMMARY.md`**: Summary of Synth Mode UI redesign (v1.0.2)
- **`UI_FINAL_UPDATES.md`**: Box character refinements and visual updates (v1.0.2)
- **`UI_LAYOUT_FIX.md`**: Vertical positioning fix for synth control boxes (v1.0.3)

## Main Documentation

For general usage and installation instructions, see the main [README.md](../README.md) in the project root.

For version history and release notes, see [CHANGELOG.md](../CHANGELOG.md) in the project root.
